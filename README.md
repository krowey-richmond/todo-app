# Todo App

A simple Todo application built with HTML, CSS, and JavaScript. It allows users to add, complete, and delete tasks with data saved in local storage.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using LocalStorage
- Responsive design

---

## Tech Stack
- HTML
- CSS
- JavaScript

---

## How to Run
1. Clone or download the project
2. Open `index.html` in a browser
3. Start adding tasks

---

## Notes
Data is stored locally in the browser using LocalStorage.
